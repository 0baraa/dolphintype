## Running the Frontend

1. Clone the repository:
   ```bash
   git clone https://github.com/0baraa/dolphintype
   cd dolphintype
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to:
   ```
   http://localhost:5173
   ```
